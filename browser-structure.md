### 瀏覽器的各結構介紹：   
:::info
    用戶界面(user interface):
這包括網址欄、上一頁/下一頁按鈕、書籤等請求頁面
:::

:::info
    瀏覽器引擎(browser engine):
負責UI和渲染引擎之間的調動
:::

:::info
    渲染引擎(rendering engine):
負責顯示用戶請求的網頁內容
:::

:::info
    網路引擎(Networking):
對於HTTP請求等網絡調用的實現
:::

:::info
    UI後端(UI backend):
用於繪製網頁的各基本小部件

:::

:::info
    JavaScript解析器(JavaScript interpreter):
解析和執行 JavaScript 代碼
:::

:::info
    數據存儲(Data storage):
這是一個持久層，幫助瀏覽器在本地保存各種數據
:::
